# 🐤 金贝贝桌面精灵

> 一个基于 ESP32-S3 的桌面机器人精灵 - 小鸡形态，可对话、可互动、可手机控制

## 📋 项目概述

**项目名称：** 金贝贝桌面精灵 (JinBeibei Desktop Pet)

**核心功能：**
- 🎨 小鸡形象动态显示（LVGL 图形库）
- 💬 大模型对话能力（接入 Qwen/本地模型）
- ⏰ 闹钟、提醒、日程管理
- 📱 手机控制（飞书机器人/Web 控制页）
- 🎵 语音交互（麦克风 + 扬声器）

**技术栈：**
- 硬件：ESP32-S3 + 4 寸触摸屏
- 固件：PlatformIO + Arduino/ESP-IDF
- UI: LVGL 8.x
- 通信：WiFi + WebSocket/MQTT/HTTP
- 移动端：Flutter（iOS/Android 跨平台）
- 后端：自建 WebSocket 服务器 + Qwen API

## 🎨 金贝贝形象设计

**小鸡特征：**
- 🐤 黄色毛绒身体
- 👀 大眼睛（可表达情绪）
- 🎤 小嘴巴（说话时动）
- 💤 会睡觉、会开心、会生气
- 🎵 听到音乐摇摆

**情绪状态：**
| 状态 | 表情 | 触发条件 |
|------|------|----------|
| 开心 | 😊 眼睛弯弯 | 收到夸奖、完成任务 |
| 困倦 | 😴 眼皮下垂 | 深夜、长时间未互动 |
| 生气 | 😠 眉毛下压 | 被批评、错误发生 |
| 好奇 | 🤔 歪头 | 收到新问题 |
| 说话 | 🗣️ 嘴巴动 | 语音回复时 |

## 📝 快速开始

### 环境要求
- VSCode + PlatformIO 插件
- Python 3.8+
- Git

### 编译烧录
```bash
# 克隆项目
git clone <repo-url>
cd jinbeibei-robot/src

# 编译
pio run

# 烧录
pio run --target upload

# 串口监视
pio device monitor
```
