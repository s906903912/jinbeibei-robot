| GPIO42 | PCLK | 像素时钟 |
| GPIO38 | R0 | 红色 bit0 |
| GPIO18 | R1 | 红色 bit1 |
| GPIO17 | R2 | 红色 bit2 |
| GPIO16 | R3 | 红色 bit3 |
| GPIO15 | R4 | 红色 bit4 |
| GPIO8 | G0 | 绿色 bit0 |
| GPIO19 | G1 | 绿色 bit1 |
| GPIO20 | G2 | 绿色 bit2 |
| GPIO21 | G3 | 绿色 bit3 |
| GPIO22 | G4 | 绿色 bit4 |
| GPIO23 | B0 | 蓝色 bit0 |
| GPIO47 | B1 | 蓝色 bit1 |
| GPIO48 | B2 | 蓝色 bit2 |
| GPIO45 | B3 | 蓝色 bit3 |
| GPIO46 | B4 | 蓝色 bit4 |
| 5V | VCC | 电源正极 |

### 音频模块连接

#### MAX98357 功放（I2S）

| ESP32-S3 | MAX98357 | 说明 |
|---------|----------|------|
| GPIO5 | BCLK | 位时钟 |
| GPIO6 | LRCLK | 左右时钟 |
| GPIO7 | DIN | 数据输入 |
| 5V | VCC | 电源 |
| GND | GND | 地 |
| GPIO4 | SD_MODE | 模式选择（上拉） |

#### INMP441 麦克风（I2S）

| ESP32-S3 | INMP441 | 说明 |
|---------|---------|------|
| GPIO5 | SCK | 时钟（与功放共用） |
| GPIO47 | WS | 字选择 |
| GPIO48 | SD | 数据输出 |
| 3.3V | VDD | 电源 |
| GND | GND | 地 |
| GND | L/R | 左声道 |

### 喇叭连接

```
MAX98357 OUT+ ────┬──── 喇叭+
                  │
MAX98357 OUT- ────┴──── 喇叭-
```

## 电源连接

```
Type-C 充电头 (5V/2A)
       │
       ▼
┌──────────────┐
│  Type-C 接口  │ → ESP32-S3 开发板
│  (开发板自带) │
└──────────────┘
       │
       ├──→ 屏幕 VCC (5V)
       ├──→ MAX98357 VCC (5V)
       └──→ INMP441 VDD (3.3V, 从 ESP32 取电)
```

## 注意事项

⚠️ **重要提醒：**

1. **电平匹配**：ESP32-S3 是 3.3V 逻辑电平，确保屏幕和模块兼容
2. **电源功率**：建议使用 5V/2A 以上电源，屏幕和功放同时工作时电流较大
3. **I2S 时钟**：麦克风和功放共用 BCLK 时需注意相位配置
4. **接地**：所有模块的 GND 必须共地
5. **信号线长度**：RGB 屏幕信号线尽量短，避免干扰
